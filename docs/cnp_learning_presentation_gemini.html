🤖 What are Microbots?

Lightweight, extensible AI agents for code comprehension and controlled file edits

🔒 Secure

Containerized execution with explicit permission controls

🎯 Controlled

Read-only or read-write modes with least-privilege access

🔧 Extensible

Custom tools and LLM provider support

🚀 Integration-Ready

Fits cleanly into automation pipelines

Example Usage:

from microbots import WritingBot myBot = WritingBot( model="azure-openai/my-gpt5", folder_to_mount="myReactApp", ) result = myBot.run("""when doing npm run build, I get an error. Fix the error and make sure the build is successful.""", timeout_in_seconds=600)

💡 Core Concepts

📖ReadingBot

  • Safe code comprehension and analysis
  • Read-only access to mounted directories
  • Perfect for debugging and error explanation

✍️WritingBot

  • Controlled code modifications
  • Read-write access with proper permissions
  • Automated fixes and refactoring

🔐Permission Model

  • Explicit mount with READ_ONLY or READ_WRITE permissions
  • Containerized environment isolation
  • Least-privilege access by design
